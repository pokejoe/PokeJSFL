PokePublish v1.0

Download
ダウンロード

Description | 説明

Publishes HTML from Flash using toolkit for CreateJS and automatically converts the HTML, replaces the files with the compressed version, and adds loader.gif. With one button press, your animation is ready to be used in Pokelabo games.

Toolkit for CreateJSをつかってFlashからHTMLをパブリッシュします。そして、自動的にHTMLを変換して、画像を軽量版で上書きして、loader.gifを書き込みます. ボタンを一つ押すだけで、アニメーションをポケラボのゲームにすぐに使えます。

  • To install, unzip the archive in the following location:
    インストール方法は、アーカイブファイルを以下のフォルダーにご解凍ください。 
    
 C:\Users\user.name\AppData\Local\Adobe\Flash CS6\ja_JP\Configuration\WindowSWF
            ^                               ^      ^
         Windows login           Flash version    　
         Windowsユーザー名　　     Flashバージョン
  • Restart Flash.　The panel can be found on the Window->Other Panels menu.
    Flashをご再起動。パネルはウィンドウ->他のパネルに見つけられます。
  • Set your output folder to 01.本番 in the Toolkit options, as normal.
    いつもの通りに、Toolkitの設定に出力フォールだーを01.本番お選びください。
  • Compress the images that will end up in 01.本番/images/ and store them in a separate folder. You may have to publish normally with Toolkit first to obtain these images.
    01.本番/images/の画像を軽量にして、別のフォルダーにご保存。画像を制作ために、Toolkitで1回普通のパブリッシュが必要かもしれません。
  • Input the title and push PokePublish!
    タイトルを入力して、PokePublish!ボタンを押すと終わりです！

Warning | 注意

Assumes you have Toolkit for CreateJS v1.2 installed. If you do not, use the web-based tool.

Toolkit for CreateJS v1.2がインストールしたことが仮定します。そうしない場合、webのツールをご利用ください。
