Unskew　v0.2

Download
ダウンロード
.

Description | 説明

In Flash, sometimes rotation keyframes accidentally become skew keyframes. But when we export them this causes problems. This script examines each frame of each layer of an opened MovieClip, detects whether skew is present on the first element, and converts it to rotation.

フラッシュで、回転のキーフラムは時々傾斜のキーフレームに誤って変換されています。書き出せば、問題になります。このスクリプトは開いているムービークリップのすべてのレイヤーの各フレームの一番最初elementを確認して、傾斜を調べます。傾斜がある場合は、回転に変換します。

Warnings | 注意

  • It will check and convert every keyframe in your movie clip. This can take a very long time if you have a lot of frames and layers. During this time, Flash will become unresponsive, but it has not crashed. You should still save your work before running it.
  • It will only convert keyframes that are displayed on the timeline. In other words, the move clip must be open - it will not remove skew keys from the interior animation of instances of movie clips.
  • This is meant to correct accidental skews that are supposed to be rotations. If you drastically skew your objects on purpose, you will end up with some funny scale/rotations.
  • 各キーフレームを確認して変換します。フレームとレイヤーが多い場合は、長い時間かかります。実行中はFlashがフリーズしますが、クラッシュじゃないです。実行前にファイルを保存した方がいいです。
  • タイムラインに表示されたキーフレームしか変換できません。言い換えると、実行前にムービークリップを開かなけれべなりません。インスタンスクリップの中に入ってあるキーフレームを変換できません。
  • 不慮作成した傾斜キーフレームだけ向けです。抜本的に傾斜すれば、スクリプトを実行するとおかしい回転とサイズの結果になります。

Installation | インストール

  1. Right click the download button and "Save As." If you just click, it will open as text in your browser.
  2. It should be saved to your Flash commands directory. Normally, it is in this location: C:\Users\joseph_jacir\AppData\Local\Adobe\Flash CS6\ja_JP\Configuration\Commands - replace the bolded portion with your own PC login.
  3. After that, the script will appear in your commands menu at the top of the Flash environment. You can run it by clicking there.
  4. You can　also assign a hotkey to it (I use Ctrl+Shift+U).

  1. 上記のボタンを右クリックして、「名前をかけて保存」クリックしてください。左クリックすれば、スクリプトはブラウザーにテキストで開きます。
  2. フラッシュのコマンドフォルダに保存した方がいいです。普通には、このフォルダになります:　C:\Users\joseph_jacir\AppData\Local\Adobe\Flash CS6\ja_JP\Configuration\Commands。太字を自分のユーザー名を変わってください。
  3. それに、スクリプトはフラッシュのコマンドのメニュに現出します。あそこにクリックすると、実行します。
  4. キーボードショートカットを充てられる。(私はCtrl+Shift+Uを使っています。)
