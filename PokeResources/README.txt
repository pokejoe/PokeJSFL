1)
Some scripts, especially ones that involve the manipulation of files, depend on the resources stored in this folder. 
あるスクリプト（特にファイル管理を対応するの）はこのフォルダーに保存されたファイルに従属します。

2)
Make sure this is a subfolder of your Flash config directory! For example:
このフォルダーはぜひフラッシュConfig保存先のサブディレクトリにしてください！たとえば
	"C:\Users\joseph_jacir\AppData\Local\Adobe\Flash CS6\ja_JP\Configuration\PokeResources"

3)
Finally, in order to be able to use file upload functions, make sure you set your user settings in LocalUserSettings.jsfl are correct!! Otherwise they may not upload, or upload into someone else's directory.
最後に、LocalUserSettings.jsflの設定をぜひ確認をお願いします！そうしないと、アップロードの失敗の可能も、ほかの人の保存先にアップの可能もありますから。
