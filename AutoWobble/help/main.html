AutoWobble

Automatically creates bouncing and wobbling keyframe animations
自動的にバウンドと揺れるアニメーションキーフレームを作成してくれるツール

Workflow　使い方 / ワークフロー

  1. Animate a primary motion - an object moving from one position, rotation, and scale to another. Leave (or create) frames afterward for the bounce.
    一次動きのアニメーションを作って。オブジェクトの位置、回転、とスケールは一つからもう一つまでのアニメーションの作成です。
    揺れのために、アニメの終わりの後余地を残して。

  2. Select the last keyframe of the motion (B), and the normal frames following it (arbitrary length)

    一次動きの最後のキーフレーム(B)と翌フレームを選択して。長さはなんでも大丈夫です。

  3. If using the GUI, adjust the parameters in the SWF panel, or choose a preset. If not using the GUI (i.e. menu or keyboard shortcut), it will use parameters from the last time it was run. This may differ from what is displayed in the GUI.

    GUIの利用場合は、引数を調整して。または、プリセットを選択して。
    GUIを利用しない場合は、前回の実行の変数を使います。
    現在のUIで表現されている値と違う可能性もありますので、ご注意ください。
  4. Execute script by pushing the "Wobble!" button, running the command, or using the keyboard shortcut.
    「揺れ！」ボタンを押すとスクリプトを実行します。または、コマンドメニュー、キーボードショートカットも利用できます。

Keyframes　キーフレーム

Interval　期間

The frame length of each extreme's keyframe.
各揺れの先端の間のフレーム数。

  • Interval = 1 期間 = 1


  • Interval = 3 期間 = 3


  • Interval = 5 期間 = 5


Bounce %　跳ね返し％

The maximum bounce amount for position, rotation/skew, scale, and stretch. These can be be controlled more finely with the individual effect % controls.

位置、回転/傾斜、スケール、と伸ばしの最高限度。別々の効果％コントロールで細かい調整もできます。

For example, if we animate position only, have position effect at 100% and Bounce % at 60:
例えば、位置の効果は100%で、跳ね返し％は60%の場合

Decay　減衰

The rate at which the object stops bouncing and returns to the target keyframe. Follows this function (drag the "a" slider).

オブジェクトのバウンドが終わって、ターゲットのキーフレームに戻す速度。
この数学関数のとおりにします。(「a」スライダーをドラッグしてください。)

Easing　イージング

All new tweens are eased by this amount. Alternates between positive and negative.
すべての新しいトウィーンはこの値でイージングを付けます。正数と負数の間を行ったり来たりします。

Effect %　効果％

Position, Rotation/Skew, Scale
位置、回転/傾斜、スケール

A further fraction of the bounce % applied to each attribute. So if bounce is 60% and position is 30%, the position will bounce a maximum of 18% of the primary motion.

各プロパティーの跳ね返し％の割合。
たとえば、跳ね返し％が60%で、位置は30%だったら、オブジェクトの位置は元の動きの18%で揺れます。
The reason to do this is that it allows you to emphasize one type of motion over another - for example, more rotation bounce than position bounce.

使う理由は、他の動きよりも一つのタイプを力説できるようになります。
例えば、位置より、回転のほうが揺れされたい場合はこんな感じで。

Stretch　伸ばし

This works similarly to other effect controls, in that it represents a fraction of the maximum bounce %. However, this is used to control squash and stretch, and is calculated as a proportion of the distance moved (as a percentage of the size of the object). It affects scale (unevenly) based on position's movement, but works in addition to the scale control. Visually, it is a cartoonish bounciness.

最大跳ね返し％の割合コントロールなので、これは上記の効果コントロールと似ています。
しかし、このコントロールは子どものアニメっぽい「squash and stretch」に対応します。　
（オブジェクトのサイズの割合で）動きの距離によって計算されます。
位置の動きによって、スケールに影響を及ぼしますが、スケールコントロールに加えて影響を及ぼします。

Presets プリセット

Because these controls can be too detailed, for convenience it is possible to save your own presets, or to load ones that I have included with the tool. Choose one from the dropdown menu to load. Enter a name into the dropdown menu and press "Save new preset" to save a new one. Press "Del" to delete the currently selected preset.

上記のコントロールは詳しすぎる可能性もありますので、プリセットを保存できます。または、私が含まれたプリセットも読み込むこともできます。
読み込むように、ドロップダウン・メニューから１行を選択して。
保存するように、ドロップダウン・メニューにプリセット名を入力して、「プリセット保存」ボタンを押して。
削除するように、プリセットを選択して、「消し」ボタンを押して。

Here are some example use cases.
以下は、例の使用事例です。

Run　実行

Wobble! 揺れ！

Press this to add the wobble keyframes. Refer to the workflow section for instructions. If an improper keyframe selection is made, you will receive an error alert.

このボタンを押すと、揺れのキーフレームを挿入します。ワークフローの指示に参考してください。
無効なフレーム範囲が選択されている場合は、エラー通知が出ます。

Revert 戻し

When using the keyboard shortcut or command menu, this button is not necessary. However, when using the GUI, Flash's undo command undoes each keyframe one at a time. This takes time and is annoying, so this is a quicker way to undo a wobble motion.

キーボードショットカットかコマンドメニューを利用の際、
このボタンは必要がありません。
しかし、GUIを利用の際は、
フラッシュの取り消しコマンドは、新しい揺れキーフレームを一つずつ次々で消します。
大変ですので、このボタンのほうが簡単で速くなります。

  1. Select the original motion's last keyframe (B) and the wobble keyframe range only. This selection should be the same frames you selected to originally create the wobble.

    一次動きの最後のキーフレーム（B）と揺れのキーフレームだけを選択して。
    この選択範囲は、最初に揺れのアニメーションを作成したときの選択範囲と同じようにください。
  2. Press the Revert button.

    戻しボタンを押して。
Caution: While this button does check for mistakes before running, it is possible to delete the wrong keyframes if your selection is wrong, so please be careful.

注意：　実行前に間違いを検出してみますのに、
相違のキーフレームを選択すれば、
揺れ以外のキーフレームを削除可能性もありますので、
ご注意ください。
