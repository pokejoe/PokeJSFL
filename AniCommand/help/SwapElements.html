Swap Elements

Select any number of keyframes on the timeline. Then select a library item. The object on each keyframe will be replaced by the selected element.

タイムラインにキーフレームをいくらでも選択して、ライブラリアイテムを一つ選択して、コマンドを実行すると、各選択されたキーフレームのオブジェクトは選択されたライブラリアイテムで取り返します。

Recommended keyboard shortcut: Ctrl+Alt+/
おすすめのキーボードショートカット　Ctrl+Alt+/
(Similar to After Effects - After Effectsと同じですから)
