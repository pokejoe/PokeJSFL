Library Name Prefix

Select one or more library items and run the command. You will be prompted for a string, which is then inserted at the beginning of the name of all selected library items.

ライブラリーアイテムを1個以上選択して、コマンドを実行すると、入力プロンプトが出てきます。
文字列を入力すると、すべての選択されたライブラリアイテム名の前に挿入します。

Recommended keyboard shortcut: Ctrl+Alt+Shift+P
おすすめのキーボードショートカット　Ctrl+Alt+Shift+P
