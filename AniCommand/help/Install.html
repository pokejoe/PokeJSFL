Installation and Setup
インストールとセットアップ

日本語

The AniCommand panel is part of the PokeAniToolbox. Download the installer. Remember to set your remote FTP folder once installed. The installer will open this setting for you.

Once installed, restart Flash. The panel is accessible under Flash's menus: Window -> Other Panels -> AniCommand.

The panel is the same size as the alignment and transform windows (整列 & 変形), so I recommend docking it there.

All commands are also available under the Command menu. Because of this, you can also assign them to keyboard shortcuts.


AniCommandというパネルは、PokeAniToolboxというパッケージの一部です。
インストーラーのダウンロードリンク。
インストール後、ぜったいリモートFTPフォルダを設定してください。
インストーラーは、終わったらこの設定を開いてくれます。

インストールしたら、フラッシュを再起動してください。
このパネルは、フラッシュのウィンドウメニューにアクセスできます。　ウィンドウ　→　ほかのパネル　→　AniCommand。　

パネルは、整列パネルと変形パネルと同じサイズですので、
おすすめのあそこにドッキング所はあそこになります。

AniCommandパネルに入っているコマンドはすべてコマンドパネルからも実行できます。
ですので、キーボードショートカットも代入できます。
